# Task1_Wepage-InAmigos

# 🌟 InAmigos Foundation — NGO Awareness Webpage

A responsive static awareness webpage created using HTML and CSS showcasing the projects, social impact, and initiatives of InAmigos Foundation — a Section 8 registered non-profit organization from Bilaspur, Chhattisgarh, India.

---

# Live Demo

## Vercel:
https://webpage-inamigos.vercel.app/

---

# 📌 Project Overview

InAmigos Foundation Awareness Page is a modern NGO-themed static webpage designed to present the foundation's ongoing projects, social impact numbers, volunteer opportunities, and contact details through an attractive and structured user interface.

This project was developed as part of the **Project Vikas — Frontend Development Internship** under InAmigos Foundation, using only HTML and CSS.

---

# ✨ Features

- Responsive webpage design
- Sticky navbar with hamburger menu for mobile
- Donate Now button with QR code popup modal
- Modern hero section with CTA buttons
- Ongoing projects section with image cards
- Social impact statistics section
- Video highlights section
- Get Involved / Volunteer section
- Footer with social media and contact links
- Clean, readable, and structured UI
- Mobile responsive design

# 🎨 Design Highlights

- Elegant typography using Google Fonts (Poppins)
- InAmigos brand color palette — Purple `#6c3fc5` and Orange `#f4793a`
- Responsive card layouts using CSS Grid
- Hover lift effects and smooth transitions on all cards
- Gradient hero and impact sections
- Modal popup with backdrop blur for donation QR code
- Dark-themed Get Involved and Footer sections

---

# 📱 Responsive Design

The website is responsive and adapts to:

- Desktop screens (full navbar, 3-column grid)
- Tablets (compact navbar, 2-column footer)
- Mobile devices (hamburger menu, stacked layout)
- Small phones (further scaled buttons and logo)

Media queries are used for smooth responsiveness across all screen sizes.
